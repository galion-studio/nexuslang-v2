# Code of Conduct - NexusLang v2

## Our Pledge

In the interest of fostering an open and welcoming environment, we as contributors and maintainers pledge to make participation in NexusLang v2 and our community a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Responsibilities

### Maintainers' Responsibilities

Project maintainers are responsible for:
- Clarifying standards of acceptable behavior
- Taking appropriate and fair corrective action
- Removing, editing, or rejecting comments, commits, code, issues, and contributions that don't align with this Code of Conduct
- Communicating reasons for moderation decisions when appropriate

### Community Members' Responsibilities

Community members are expected to:
- Follow this Code of Conduct
- Report violations
- Support a welcoming environment
- Help newcomers
- Be professional in all interactions

## Scope

This Code of Conduct applies:
- Within project spaces (GitHub, Discord, etc.)
- In public spaces when representing the project
- When using official project accounts
- At project events and meetups

---

## Enforcement

### Reporting

**If you experience or witness unacceptable behavior:**

1. **For immediate safety concerns:** Contact conduct@galion.app
2. **For other violations:** Report via GitHub or email

**Your report will include:**
- Your contact information
- Details of the incident
- Additional context if relevant

**We promise:**
- Confidentiality
- Fair investigation
- Appropriate response

### Response Process

1. **Report received** - Within 24 hours: Acknowledgment
2. **Investigation** - Within 7 days: Review evidence
3. **Decision** - Fair and appropriate action
4. **Communication** - Parties notified of outcome

### Consequences

Violations may result in:

**First Offense:**
- Private warning
- Public apology requested
- Temporary ban (if severe)

**Second Offense:**
- Temporary ban (1 month)
- Required apology
- Probation period

**Third Offense:**
- Permanent ban from project
- Removed from contributor list

**Immediate Ban:**
- Severe harassment
- Threats of violence
- Doxxing or privacy violations

## Community Guidelines

### Communication

**Be Clear:**
- Use simple language
- Explain technical terms
- Provide context
- Share links to relevant docs

**Be Respectful:**
- Disagree politely
- Avoid absolutes ("always", "never")
- Acknowledge others' points
- Stay on topic

**Be Helpful:**
- Answer questions thoroughly
- Share resources
- Encourage learning
- Celebrate successes

### Contributing Code

**Quality Standards:**
- Code must pass tests
- Follow style guidelines
- Include documentation
- Add examples if applicable

**Review Process:**
- Be constructive in reviews
- Suggest improvements, don't just criticize
- Explain reasoning
- Approve when ready
