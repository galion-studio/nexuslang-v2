üëã Welcome to Nexus Core API

Nexus Core is a modern microservices platform built with Go and Python. This documentation provides complete API references for all services in the Nexus ecosystem.

Architecture: The system consists of an API Gateway that routes requests to specialized microservices. Each service handles a specific domain (authentication, users, analytics, etc.) and communicates via REST APIs and Kafka events.

Authentication: Most endpoints require JWT tokens. Obtain a token by registering and logging in through the Auth Service.

üóÑÔ∏è Infrastructure Services

Nexus Core runs on a robust infrastructure stack:

  • PostgreSQL (Port 5432) - Primary database for all services
  • Redis (Port 6379) - Caching and rate limiting
  • Kafka (Port 9092) - Event streaming and messaging
  • Prometheus (Port 9091) - Metrics collection and storage
  • Grafana (Port 3000) - Visualization and dashboards
  • Kafka UI (Port 8090) - Kafka management interface
