# GALION.STUDIO Frontend

React app for the GALION.STUDIO Alpha MVP.

## Quick Start

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Make sure backend is running first!

```bash
cd ..
python app.py
```

Backend must be on port 5000 for proxy to work.

## Build for Production

```bash
npm run build
```

Outputs to `build/` folder.

## Tech Stack

- React 18
- react-beautiful-dnd (drag & drop)
- axios (API calls)
- Pure CSS (no Tailwind, keeping it simple)

Built following Musk's principles: Simple. Fast. Effective.
