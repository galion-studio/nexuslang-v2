## üí° TIPS FOR BEST RESULTS

### ‚úÖ DO:
- **Speak clearly** at normal pace
- **Use natural language** (don't over-formalize)
- **Provide context** when needed (e.g., "search for recent AI articles")
- **Wait for response** before next command
- **Use push-to-talk** for best accuracy

### ‚ùå DON'T:
- Speak too fast or mumble
- Use very long sentences (keep under 30 seconds)
- Include background noise
- Interrupt while AI is speaking
- Use slang or abbreviations heavily

## üéØ COMMAND PATTERNS

### Basic Pattern
```
[Action] [Target] [Parameters]

Examples:
- "Show my profile" (action: show, target: profile)
- "Search users named John" (action: search, target: users, param: John)
- "Create post about AI" (action: create, target: post, param: AI)
```

### With Entities
```
[Action] [Entity Type] [Entity Value]

Examples:
- "Login as john@example.com" (action: login, entity: email)
- "Update name to Sarah" (action: update, entity: name, value: Sarah)
```

### Complex Pattern
```
[Action] [Target] [Condition] [Parameters]

Examples:
- "Search users with role admin"
- "Show posts tagged with Python created last week"
- "Find articles about AI with more than 100 views"
```

## üîß TROUBLESHOOTING

### Voice Not Working
1. Check microphone permissions
2. Verify internet connection
3. Try text fallback
4. Refresh page
5. Clear browser cache

### Poor Accuracy
1. Reduce background noise
2. Speak more clearly
3. Use push-to-talk
4. Try shorter commands
5. Use text for complex commands

### Slow Response
1. Check internet speed
2. Try during off-peak hours
3. Use text for faster response
4. Close other tabs/apps
