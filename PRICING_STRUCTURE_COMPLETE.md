## ğŸ”— DEVELOPER PLATFORM DEEP DIVE

### API Documentation Structure

**Base URL**: `https://api.developer.galion.app`

**Authentication**:
```http
Authorization: Bearer YOUR_API_KEY
```

**Get API Key**:
1. Register at developer.galion.app
2. Go to Dashboard â†’ API Keys
3. Generate new key
4. Copy and use in headers

### Example API Calls:

**1. Execute NexusLang Code:**
```bash
curl -X POST https://api.developer.galion.app/api/v2/nexuslang/run \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "fn main() { print(\"Hello!\") }\nmain()",
    "compile_to_binary": false
  }'

# Response:
{
  "output": "Hello!",
  "execution_time": 67.2,
  "success": true,
  "credits_used": 1
}
```

**2. Generate Image:**
```bash
curl -X POST https://api.developer.galion.app/api/v2/ai/generate/image \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A futuristic AI laboratory",
    "model": "stable-diffusion-xl",
    "size": "1024x1024"
  }'

# Response:
{
  "image_url": "https://cdn.galion.app/images/abc123.png",
  "model": "stable-diffusion-xl",
  "credits_used": 5
}
```

**3. Generate Video:**
```bash
curl -X POST https://api.developer.galion.app/api/v2/ai/generate/video \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "AI robot walking through a city",
    "duration": 5,
    "model": "runway-gen3"
  }'

# Response:
{
  "video_url": "https://cdn.galion.app/videos/xyz789.mp4",
  "duration": 5.0,
  "credits_used": 20
}
```

**4. AI Chat (via OpenRouter):**
```bash
curl -X POST https://api.developer.galion.app/api/v2/ai/chat \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [{"role": "user", "content": "Explain quantum computing"}],
    "model": "anthropic/claude-3.5-sonnet",
    "max_tokens": 1000
  }'

# Response:
{
  "content": "Quantum computing is...",
  "model": "claude-3.5-sonnet",
  "tokens_used": 324,
  "credits_used": 1
}
```
