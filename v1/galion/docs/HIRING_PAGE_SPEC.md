## MISSION

Build a hiring page that:
- Attracts top talent with radical transparency
- Shows compensation ranges (no secret salaries)
- Highlights our values (first principles, speed, transparency)
- Tracks analytics (applications, conversions, quality)

- Transparency attracts the right people
- Clear expectations reduce mismatches
- Data-driven hiring improves quality

## DATABASE SCHEMA

```sql
CREATE TABLE applications (
  id UUID PRIMARY KEY,
  position_id UUID REFERENCES positions(id),
  name VARCHAR(255) NOT NULL,
  email VARCHAR(255) NOT NULL,
  location VARCHAR(255),
  timezone VARCHAR(50),
  linkedin_url VARCHAR(500),
  github_url VARCHAR(500),
  portfolio_url VARCHAR(500),
  years_experience INT,
  expected_salary INT,
  available_start_date DATE,
  why_join TEXT,
  project_description TEXT,
  status VARCHAR(50) NOT NULL,  -- pending, reviewed, interview, offer, accepted, rejected
  reviewed_by UUID REFERENCES users(id),
  reviewed_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE positions (
  id UUID PRIMARY KEY,
  title VARCHAR(255) NOT NULL,
  description TEXT,
  requirements TEXT,
  salary_min INT,
  salary_max INT,
  location VARCHAR(255),  -- remote, hybrid, onsite
  employment_type VARCHAR(50),  -- full-time, part-time, contract
  status VARCHAR(50) NOT NULL,  -- open, closed, filled
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE interviews (
  id UUID PRIMARY KEY,
  application_id UUID REFERENCES applications(id),
  interviewer_id UUID REFERENCES users(id),
  type VARCHAR(50) NOT NULL,  -- technical, culture
  scheduled_at TIMESTAMP,
  duration_minutes INT,
  notes TEXT,
  rating INT,  -- 1-5
  decision VARCHAR(50),  -- pass, fail
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE offers (
  id UUID PRIMARY KEY,
  application_id UUID REFERENCES applications(id),
  salary INT NOT NULL,
  equity DECIMAL(5,2),
  start_date DATE,
  status VARCHAR(50) NOT NULL,  -- pending, accepted, rejected
  sent_at TIMESTAMP,
  responded_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);
```
